Разработана интерактивная 2D игра на основе Canvas с поддерживающей веб-платформой, включающей чат в реальном времени и систему рейтинга игроков. Реализован бэкенд с использованием Nest.js с надежными API-эндпоинтами и JWT-аутентификацией, обеспечивая безопасный поток авторизации на стороне клиента. Приложение включало комплексное покрытие модульными тестами с использованием Jest для валидации основной игровой логики, API-сервисов и middleware аутентификации, поддерживая покрытие тестами 85%+ на протяжении разработки.

## Технологии

- **Frontend**: React, Ant Design, Canvas API, Redux-Toolkit, RTK Query
- **Backend**: Nest.js с TypeScript
- **Тестирование**: Jest, Supertest
- **Аутентификация**: JWT-безопасность

## Обязанности

- Разработка фронтенда с React и Canvas API
- Разработка бэкенд API с Nest.js
- Лидерство в проекте и архитектурные решения
- Комплексное модульное тестирование (покрытие 85%+)

## Ключевые возможности

- Механика многопользовательской игры в реальном времени
- Система живого чата для игроков
- Система рейтинга и таблицы лидеров игроков
- Безопасная JWT-аутентификация
- Высокое покрытие тестами, обеспечивающее качество кода
