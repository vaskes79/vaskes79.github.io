Проект для создания заметок в видео и последующего экспорта в различных форматах разметки (HTML, Markdown, текст). Экспортированные заметки включают URL с временной позицией и описание фрагмента видео. Все заметки хранились в хранилище IndexedDB. В интерфейсе пользователи могли выполнять CRUD-операции с заметками. Я не сохранял данные во внешнем хранилище, все данные сохранялись в браузере пользователя. Если пользователь очищает кеш данных, все данные теряются.

## Технологии

- **Frontend**: React с компонентами Material-UI
- **Хранилище**: IndexedDB для клиентского хранения данных
- **Развертывание**: Netlify для хостинга
- **Форматы экспорта**: HTML, Markdown, обычный текст

## Возможности

- Создание, чтение, обновление и удаление видео-заметок
- Экспорт заметок в нескольких форматах (HTML, Markdown, текст)
- Видео URL с временными метками для быстрой навигации
- Клиентское хранилище с использованием IndexedDB
- Отсутствие зависимостей от внешнего сервера

## Ключевые детали реализации

Все данные хранятся локально в браузере пользователя с использованием IndexedDB, обеспечивая решение, ориентированное на конфиденциальность. Приложение позволяет пользователям аннотировать видео заметками с временными метками, которые можно экспортировать для обмена или документирования.
