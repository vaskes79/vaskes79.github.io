Разработана панель управления актерами, позволяющая агентам по талантам безопасно вводить и управлять профилями исполнителей. Система использовала MongoDB для структурированного хранения данных, в то время как медиа-ресурсы были выгружены в Cloudflare для оптимизированной доставки. Каждая новая заявка актера автоматически запускала процесс пересборки, генерируя обновленный статический контент, который развертывался бесшовно в AWS S3 для мгновенной доступности.

Построенная с доступностью как основным принципом, панель управления реализовала стандарты WCAG 2.1 AA во всем интерфейсе администратора. Проведено обширное тестирование с скринридерами с использованием NVDA и VoiceOver, обеспечивая полную навигацию с клавиатуры и правильную маркировку ARIA. Сотрудничал со специалистами по доступности для валидации реальной удобности использования, интегрируя автоматизированные проверки a11y в CI/CD пайплайн для поддержания соответствия во всех обновлениях.

## Технологии

- **Архитектура**: JAMstack
- **Backend**: Strapi CMS
- **Frontend**: GatsbyJS
- **База данных**: MongoDB
- **Хранилище**: Cloudflare для медиа-ресурсов
- **Развертывание**: AWS S3
- **CI/CD**: Travis CI для развертывания и тестирования

## Обязанности

- Разработка фронтенда с GatsbyJS
- Настройка бэкенда с Strapi
- Конфигурация CI/CD пайплайна
- Автоматизация развертывания в AWS

## Ключевые возможности

- Безопасное управление профилями актеров
- Автоматическая регенерация статического сайта при обновлении контента
- Соответствие доступности WCAG 2.1 AA
- Комплексное тестирование с скринридерами (NVDA, VoiceOver)
- Автоматизированные проверки доступности в CI/CD
- Оптимизированная доставка медиа через Cloudflare
- Бесшовное развертывание в AWS S3
