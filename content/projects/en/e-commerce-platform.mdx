Developed an interactive 2D canvas-based game with a supporting web platform featuring real-time chat and player ranking systems. Implemented the backend using Nest.js with robust API endpoints and JWT-based authentication, while ensuring secure client-side authorization flow. The application included comprehensive unit testing coverage using Jest to validate core game logic, API services, and authentication middleware, maintaining 85%+ test coverage throughout development.

## Technologies

- **Frontend**: React, Ant Design, Canvas API, Redux-Toolkit, RTK Query
- **Backend**: Nest.js with TypeScript
- **Testing**: Jest, Supertest
- **Authentication**: JWT-based security

## Responsibilities

- Frontend development with React and Canvas API
- Backend API development with Nest.js
- Project leadership and architecture decisions
- Comprehensive unit testing (85%+ coverage)

## Key Features

- Real-time multiplayer game mechanics
- Live chat system for players
- Player ranking and leaderboard system
- Secure JWT-based authentication
- High test coverage ensuring code quality
